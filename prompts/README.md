# Prompt Pack: 三喵聊俄國勝利日閱兵

This folder stores one slide-design prompt per slide. This version is optimized for an explanatory presentation: clear argument, visual evidence, and later editable Traditional Chinese typography.

Recommended use:
- Generate each slide as a 16:9 visual layout/background with diagram areas, panels, and evidence-style imagery.
- Add all Traditional Chinese titles, bullets, labels, and captions in PPT afterward as editable text.
- Do not ask the image model to render Chinese text directly; it should create layout blocks, visual metaphors, icons, timelines, maps, and negative space.
- Keep each prompt file independent so a single slide can be revised without touching the whole deck.

Style baseline:
- Presentation-first, not poster-first.
- 圖文並茂：一半用於清晰文字資訊，一半用於圖像、時間線、地圖、流程圖或證據拼貼。
- Tone: analytical, sober, historical, documentary.
- Visual language: clean editorial infographic, restrained documentary imagery, timeline/map/card structures, muted red/gray/black/steel palette.
- Text language: Traditional Chinese only.
